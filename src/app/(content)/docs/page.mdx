# easyrice CLI

The easyrice CLI is a tool for managing and applying your Linux desktop rice configurations with ease.

## Overview

easyrice makes it simple to:

- Browse and discover community rice configurations
- Apply rices to your system with a single command
- Manage your dotfiles and configuration backups
- Share your own rices with the community

## Quick Start

```bash
easyrice init
```

Head over to [Installation](/docs/installation) to get started, or check the full [Commands](/docs/commands) reference.
